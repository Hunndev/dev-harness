# 스코프 + 이해관계자 맵

기획의 범위와 영향받는 이해관계자를 정의한다.

## 실행 방식

이 skill은 Fork에서 실행된다.

## 절차

1. 사용자가 제시한 방향을 분석한다.
2. 범위를 정의한다:
   - IN: 이번 기획에서 결정할 것
   - OUT: 이번에 결정하지 않을 것 (의도적 제외)
   - DEPENDS: 다른 결정에 의존하는 것
3. 이해관계자를 식별한다.
4. 모호한 논의점을 정리하여 사용자에게 제시한다.

## 산출물: scope.md

```markdown
# 스코프

## 기획 주제
(한 문장 요약)

## IN (이번에 결정할 것)
- ...

## OUT (이번에 결정하지 않을 것)
- ... — 이유: ...

## DEPENDS (선행 결정 필요)
- ... — 의존 대상: ...
```
