# 유사 구현 조사

기존 코드베이스에서 재사용 가능한 패턴과 충돌 가능성을 조사한다.

## 실행 방식

이 skill은 Sub-agent가 자동으로 실행한다.

## Sub-agent 프롬프트

```
다음 요구사항을 바탕으로, 기존 코드베이스에서 유사 구현을 조사하라.

조사 항목:
1. 기존 코드에 비슷한 패턴이 이미 있는가? (모델 구조, API 패턴, 비즈니스 로직)
2. 기존 모델/시리얼라이저/뷰를 확장하여 재사용할 수 있는가?
3. 기존 코드와 충돌할 가능성이 있는 부분은? (URL 충돌, 모델 필드 충돌, signal 간섭)
4. 기존 테스트 패턴 중 참고할 만한 것은?

[requirements.md]
[module-registry.yaml]
[관련 모듈의 기존 코드]
```
